



# 产品价格



目前，文件存储采用按容量计费的策略，各类型文件存储价格如下。

## 容量型价格

容量型文件存储采用阶梯计费策略。

| 容量范围   | 国内机房单价(元/GiB/月) |海外机房单价(元/GiB/月) |
| ------ | ------------- |------------- |
| <= 51200 GiB | 0.2          |0.35          |
| > 51200 GiB | 0.12          |0.30        |

**注：容量型购买起步为 500GiB，步长 100GiB。**

## SSD性能型价格

| 容量范围   | 国内机房单价(元/GiB/月)            |海外机房单价(元/GiB/月)    |
| ------ | ------------------------ |-------------- |
| 任意容量大小 | 0.8 |暂未支持  |

**注：性能型购买起步为 100GiB，步长为 100GiB。**
