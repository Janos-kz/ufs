



# 产品价格



目前，文件存储采用按容量计费的策略，各类型文件存储价格如下。

## 容量型价格

#### NFSv4 协议

| 容量范围   | 国内机房单价(元/G/月) |海外机房单价(元/G/月) |
| ------ | ------------- |------------- |
| 1T-50T  | 0.2          |0.35          |
| 50T及以上 | 0.12          |0.30        |

#### NFSv3 协议

| 容量范围   | 国内机房单价(元/G/月) |海外机房单价(元/G/月) |
| ------ | ------------- |------------- |
| 1T-4T  | 0.36          |   暂无      |
| 5T-49T | 0.34          |   暂无       |
| 50T及以上 | 0.32          |   暂无    |

**注1：NFSv3 协议不再支持创建新的实例(已创建的实例仍被支持)，建议使用 NFSv3 协议的文件系统迁移至使用 NFSv4 的文件系统。**

**注2：容量型购买起步为 500GB，步长 100GB。**
